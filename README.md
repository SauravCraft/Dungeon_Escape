# Dungeon Escape 🎮

Dungeon Escape is a simple Unreal Engine C++ project built while learning core gameplay programming concepts such as actor components, movement systems, and interaction mechanics.

The project focuses on understanding how gameplay systems work internally in Unreal Engine using C++ rather than relying only on Blueprints.

---

## 🧠 What This Project Demonstrates

This project demonstrates several important Unreal Engine programming concepts:

* Creating custom **Actor Components**
* Implementing **movement logic using interpolation**
* Using **C++ gameplay classes**
* Understanding **component hierarchies**
* Working with **Unreal Engine build system**
* Structuring a **C++ Unreal project**

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/Saurav170103/Dungeon_Escape.git
```

2. Open the project

Open the `.uproject` file with Unreal Engine.

3. Build the project

If prompted, allow Unreal Engine to compile the C++ code using Visual Studio.

## 🧩 Key Gameplay Systems

### Mover Component

Handles movement of actors using interpolation and tick updates.

Example logic:

```
FVector Direction = TargetLocation - StartingLocation;
```

### Lock System

Simple gameplay interaction system demonstrating component-based design.
